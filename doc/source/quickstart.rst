PICRUST Quickstart Guide
========================

.. include:: global.rst

1. Download PICRUST Software
----------------------------
* Download the `PICRUST development software`_ using SVN 

2. Download PICRUST Data Files
------------------------------

* Download and unpack PICRUST data files (Note: Not yet available). 
* **Temporary during testing**: Instead download `PICRUST temporary files`_, which will be required for a temporary initial first step outlined below.

3. Prepare your OTU table
-------------------------

* Download the `PICRUST GG reference data`_
* Your OTU table should be created using the :ref:`otu_picking_tutorial`.

4. Run PICRUST
--------------

* Now that the hard stuff is done simply follow the :ref:`metagenome_prediction_tutorial`.
